# 耳朵阅读
耳朵阅读是一款解决碎片化时间的应用程序。

## 简介
- 支持稍后阅读：可以利用很短的时间将自己感兴趣的文章收集起来
- 支持语音播放：语音播放文章可以释放双眼，让我们在消费网络文章时，同时并行处理其他事情
- 支持多个平台：因为我们获取资讯的平台有很多，软件最好完美适配主流的资讯平台，保证用户的体验。
- 更多功能，尽情期待。
